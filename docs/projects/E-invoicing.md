# 🌟 Key Achievements:

* Successfully designed and executed comprehensive test strategies covering both manual and automation testing for the entire E-Invoicing workflow
* Developed and maintained automated test scripts using Selenium, TestNG, and Postman reducing regression testing time by 40%
* Identified and documented 50+ critical defects during manual exploratory testing, ensuring system stability before production deployment
* Created detailed test cases for all five stages of the E-Invoicing process, achieving 95% test coverage
* Implemented API automation testing for the ERP-Government portal integration, validating JSON structures, response handling, and error scenarios
* Performed end-to-end testing of the automatic QR code and IRN generation and attachment process
* Collaborated with development team to troubleshoot complex integration issues, improving system reliability
* Conducted performance testing to ensure system handles peak invoice generation loads efficiently
